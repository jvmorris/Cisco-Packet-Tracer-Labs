<h1>Cisco Packet Tracer Labs</h1>

<h2>Description</h2>

This repository contains hands-on Cisco Packet Tracer labs designed to demonstrate 
core networking concepts, device configuration, troubleshooting methodology, 
and enterprise network design principles.

Each lab simulates real-world networking scenarios including routing, switching, 
VLAN segmentation, and security implementation.

<br />

<h2>Utilities Used</h2>

- <b>Cisco Packet Tracer</b>

<br />

<h2>Environments Used</h2>

- <b>Windows 10</b> (22H2)

<br />

<h2>Lab List</h2>

<h3>Lab 01: Connect Multiple PC to Switch</h3>

<b>Objective:</b>  
Design and implement a simple Layer 2 Ethernet LAN in Cisco Packet Tracer by connecting two PCs to a switch, assigning appropriate IP addresses, and verifying end-to-end connectivity using ping.
